# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful, welcoming and constructive. Harassment and discrimination of any
kind are not tolerated. Maintainers may remove, edit or reject contributions and comments
that violate this code, and may ban contributors for behavior they deem inappropriate.

Report concerns by opening a private issue or contacting the maintainer listed on the
repository profile. All reports will be handled confidentially.
